FinPulse — Wealth Wellness Hub

FinPulse unifies traditional and digital assets into a single, real-time financial health monitor. Unlike conventional portfolio trackers, FinPulse introduces a proprietary Wealth Wellness Score — a composite metric built from three dimensions: Diversification (asset class and geographic spread), Liquidity (realisable value across 7/30/90-day horizons), and Behavioural Resilience (transaction pattern analysis to detect panic-selling, over-trading, and idle capital accumulation).

The platform aggregates data via SGFinDex-compatible open banking APIs, read-only blockchain wallet integration (no custody, no private key access), and manual input for private and alternative assets. An AI-powered scenario engine runs stress tests and delivers plain-language, personalised action plans. Per-action alert thresholds let users set live triggers — notifications fire as toasts and are managed through a central bell panel.

A built-in Community Forum connects verified industry professionals — analysts, portfolio managers, and advisers — for real-time trend discussion. AI analyst personas engage with posted threads using the Claude API.

Built on React, FastAPI, and the Claude API. Live crypto data via CoinGecko. Regulatory architecture grounded in MAS open banking and digital asset guidelines. Target market: Singapore's 240,000+ HNW individuals and the wealth advisers who serve them, operating across a $5 trillion AUM ecosystem.
